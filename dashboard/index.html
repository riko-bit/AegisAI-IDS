📌 System Overview

This AI-based Intrusion Detection System (IDS) monitors real-time network traffic, detects anomalies using machine learning models, and assigns risk levels based on behavior.

⚙️ Tech Stack

  • Backend: FastAPI (Python)
  • ML Models: Autoencoder + Random Forest
  • Realtime Capture: PyShark
  • Frontend: HTML + Chart.js

🧠 Detection Logic

  • Traffic is captured in real-time
  • Features are extracted from packets
  • Autoencoder calculates anomaly score
  • Classifier predicts attack type
  • Risk score is computed and alerts are generated
